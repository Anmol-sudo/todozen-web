# **App Name**: TodoZen

## Core Features:

- Task Input: Allow users to input and create new tasks with descriptions and due dates.
- Task List: Display tasks in a clear, aesthetically pleasing list with visual cues for priority and due dates.
- Focus Mode: Allow users to select a task to focus on, hiding other tasks and distractions.
- Task Completion: Mark tasks as complete, with a satisfying visual confirmation.
- Intelligent Prioritization Tool: Uses generative AI to analyze task descriptions and deadlines, then recommend task priorities using a dynamic tool.

## Style Guidelines:

- Primary color: Calm sky blue (#87CEEB) to promote focus and serenity.
- Background color: Light desaturated blue (#E6F3FF) to provide a gentle backdrop.
- Accent color: Soft lavender (#E6E6FA) to highlight interactive elements.
- Body and headline font: 'PT Sans' for a modern, readable, and slightly warm aesthetic.
- Use minimalist icons to represent task categories and status, ensuring clarity without visual clutter.
- Employ a clean, spacious layout with generous use of white space to enhance readability and reduce distractions.
- Use subtle, smooth animations for task transitions and completion confirmations to provide delightful feedback without being intrusive.
